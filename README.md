# Site Mostruário de Serviços | Trabalho 2º módulo ADS

Projeto front-end desenvolvido para apresentar serviços de forma moderna, visual e responsiva.  
Ideal para profissionais autônomos ou portfólio digital.

## Sobre o Projeto

O **Site Mostruário de Serviços** foi criado com foco em design limpo, navegação intuitiva e apresentação clara dos serviços oferecidos.

O objetivo principal é transformar visitantes em potenciais clientes através de uma interface atrativa e profissional.
